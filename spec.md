# 개인 홍보 사이트 제작 스펙

## 목표

나를 소개하고 홍보할 수 있는 세련된 개인 웹사이트를 만든다.
전체적인 느낌은 **애플 공식 홈페이지처럼 깔끔하고 고급스럽고, 여백이 넓고, 부드러운 애니메이션이 있는 스타일**을 원한다.

나는 코딩을 못하므로, 완성된 형태로 바로 실행 가능한 웹사이트를 만들어줘.

---

## 전체 스타일 방향

* 메인 컬러: **파란색**
* 디자인 키워드:

  * Apple-like
  * 프리미엄
  * 미니멀
  * 넓은 여백
  * 큰 타이포그래피
  * 부드러운 스크롤 애니메이션
  * 카드형 레이아웃
  * 반응형 웹
* 배경:

  * 기본은 흰색 또는 아주 밝은 블루톤
  * 섹션별로 은은한 그라데이션 사용
* 포인트 컬러:

  * Blue 계열
  * 예: `#007AFF`, `#0A84FF`, `#EAF4FF`

---

## 메뉴 구조

상단 네비게이션 메뉴는 다음 4개로 구성한다.

1. 홈
2. 자기소개
3. 포트폴리오
4. 취미

상단 메뉴는 스크롤해도 고정되도록 한다.
모바일에서는 햄버거 메뉴로 변경한다.

---

## 페이지 구성

### 1. 홈 섹션

첫 화면은 강렬하고 세련된 히어로 섹션으로 만든다.

포함 내용:

* 큰 제목

  * 예시: `안녕하세요, 저는 정현태입니다.`
* 짧은 소개 문구

  * 예시: `아이디어를 현실로 만들고, 경험을 가치로 바꾸는 사람입니다.`
* CTA 버튼 2개

  * `포트폴리오 보기`
  * `자기소개 보기`
* 배경에는 파란색 계열의 부드러운 그라데이션 또는 빛 번짐 효과를 넣는다.
* Apple 홈페이지처럼 스크롤 시 텍스트와 카드가 자연스럽게 등장하는 효과를 넣는다.

---

### 2. 자기소개 섹션

자기소개에는 다음 내용을 넣을 수 있게 구성한다.

#### 포함 요소

* 내 사진 영역

  * 이미지 파일을 쉽게 교체할 수 있도록 `profile.jpg` 같은 파일명을 사용한다.
  * 사진은 둥근 모서리 카드 또는 원형 이미지로 표시한다.
* 자기소개 문장

  * 임시 문구를 넣어두고 나중에 수정하기 쉽게 한다.
* 이력 영역

  * 타임라인 형태로 만든다.
  * 예시는 아래처럼 넣어둔다.

#### 이력 예시

* 2026 - 개인 브랜딩 웹사이트 제작
* 2025 - 다양한 프로젝트 경험 축적
* 2024 - 새로운 기술과 도구 학습
* 2023 - 커리어 성장 시작

이력은 나중에 쉽게 수정할 수 있게 코드에 주석을 달아준다.

---

### 3. 포트폴리오 섹션

포트폴리오는 총 **4개**를 보여준다.

각 포트폴리오는 카드 형태로 구성한다.

#### 카드 구성

각 카드에는 다음 요소가 들어간다.

* 프로젝트 제목
* 짧은 설명
* 사용 기술 또는 역할
* 이미지 또는 썸네일 영역
* 자세히 보기 버튼

#### 포트폴리오 예시

1. 포트폴리오 프로젝트 1

   * 설명: 첫 번째 프로젝트 설명을 입력하는 영역

2. 포트폴리오 프로젝트 2

   * 설명: 두 번째 프로젝트 설명을 입력하는 영역

3. 포트폴리오 프로젝트 3

   * 설명: 세 번째 프로젝트 설명을 입력하는 영역

4. 포트폴리오 프로젝트 4

   * 설명: 네 번째 프로젝트 설명을 입력하는 영역

카드는 마우스를 올리면 살짝 떠오르고 그림자가 생기게 한다.

---

### 4. 취미 섹션

취미는 총 **3개**를 보여준다.

각 취미도 카드 형태로 만든다.

#### 취미 카드 구성

* 취미 이름
* 짧은 설명
* 아이콘 또는 이미지
* 파란색 포인트 디자인

#### 취미 예시

1. 취미 1

   * 설명: 첫 번째 취미 설명

2. 취미 2

   * 설명: 두 번째 취미 설명

3. 취미 3

   * 설명: 세 번째 취미 설명

---

## 디자인 디테일

### Apple 느낌을 위한 요구사항

* 폰트는 깔끔한 산세리프 사용

  * 예: `Pretendard`, `Inter`, `system-ui`
* 큰 제목은 굵고 크게
* 섹션 간 여백은 넉넉하게
* 버튼은 둥근 pill 형태
* 카드에는 부드러운 그림자와 blur 효과
* 배경에는 아주 은은한 glassmorphism 효과 사용 가능
* 스크롤 시 요소가 아래에서 위로 부드럽게 등장
* 전체적으로 과하지 않고 고급스럽게

---

## 반응형 요구사항

반드시 PC, 태블릿, 모바일에서 자연스럽게 보여야 한다.

* PC: 넓은 그리드 레이아웃
* 태블릿: 2열 카드 구성
* 모바일: 1열 카드 구성
* 모바일 메뉴는 햄버거 버튼 사용

---

## 기술 요구사항

가능하면 아래 중 하나로 만들어줘.

### 우선순위 1

* HTML
* CSS
* JavaScript

초보자가 파일을 열어서 바로 확인할 수 있도록 만들어줘.


## 중요 조건

* 나는 코딩을 못하므로 설명을 쉽게 해줘.
* 복사해서 바로 실행 가능한 코드로 만들어줘.
* 디자인은 평범한 자기소개 사이트가 아니라, 애플 홈페이지처럼 고급스럽고 세련된 느낌으로 만들어줘.
* 파란색을 메인 컬러로 사용해줘.
